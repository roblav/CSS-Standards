This style guide is a compilation of suggested standards and ideas from around the internet, and where possible information has been credited.

The purpose of this guide is to provide a common standard by which the System Seed development team will work with CSS. The guide has been produced with a slant toward using Drupal coding standards.

This is a live document and subject to modification.
